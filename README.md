# ATP-AideTonProchain
Plateforme d’entraide locale à Asnières – projet A.T.P

Le projet est pour le moment en privé vous pouvez consulter et modifier vos codes entre vous je vais meettre une structure qui va vous permettre d'avoir un code de base pour éviter de toujours tout recommencer a chaque doute vérifier avec l'ia avant de push vos modifications.

Les choix de codes sont larges, pour le frontend soit l'apparance du site je le veux en React.js ou Blade recommandé mais si possible éviter le HTML et CSS
Le plus important le backend sécurisé je le veux en Node.js c'est à dire JS + Mysql, base de donnée simple pour le moment. Je m'occuperai de la sécurité et l'ajout de tables de la data base
