## Ajout de documentation

Pour ajouter de nouveaux documents :
1. Créez un fichier Markdown dans ce dossier
2. Utilisez un nommage clair et descriptif
3. Mettez à jour ce README pour référencer le nouveau document
4. Ajoutez les images associées dans le dossier `images/`
